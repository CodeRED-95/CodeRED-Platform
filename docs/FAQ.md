# FAQ

## ¿Qué es CodeRED Platform?

Plataforma modular Laravel para administrar y consultar agencias de Shalom.

## ¿Dónde está la documentación completa?

En `docs/`.

## ¿Qué base de datos usa?

PostgreSQL.

## ¿Qué usa para caché y colas?

Redis.

## ¿El proyecto tiene Docker?

Sí, con `docker-compose.yml`.

## ¿La importación está lista?

Existe la base funcional para el Gist, pero dependencias y runtime deben verificarse en Docker real.

## ¿Hay módulo de DNI o RUC?

No todavía.
