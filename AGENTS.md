# Instrucciones del proyecto

Después de cada cambio:

1. Ejecutar las pruebas relacionadas.
2. Ejecutar Laravel Pint.
3. Ejecutar PHPStan o Larastan si está disponible.
4. Compilar el frontend si se modificó CSS, JavaScript o Livewire.
5. No usar migrate:fresh.
6. No eliminar volúmenes Docker.
7. No modificar pruebas válidas solo para hacerlas pasar.
8. Informar cualquier prueba que no pueda ejecutarse.
